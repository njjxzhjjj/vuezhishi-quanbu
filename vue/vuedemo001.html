vue 的插值表达式 只能写在文本里，标签的属性中不生效

{{title}}

vue 的插值表达式 特性1 他可以 请用 js中的属性或者方法，如：字符串的长度为：{{title.length}}

vue 的插值表达式 特性2 他可以 做加减乘除 如：人民币{{rmb}}分等于{{rmb/100}}元

vue 的插值表达式 特性3 他可以 做三元运算，很重要 比如：身高{{height}}是 :{{height>=180 ? "男神":"男人"}}

问题:请把当为男神的时候变为红色，男人的时候变为绿色

如果想要改变标签属性中的文本的数据，需要绑定，这个绑定叫做指令: v-bind缩写是 :

v-bind 1、可以绑定 数据 2、可以绑定 样式 1、class 2、style 3、数组 或者对象 样式的绑定

数据绑定可以是 双向的 简称 双向绑定

首先看数据的单项绑定 使用 加法 来计算

单项绑定

+=
+={{x+y}}

单项绑定 使用插值表达式 获取结果 观察

x:{{x}} |||| y:{{y}} ||||| z:{{z}}

首先看数据的双向绑定 使用 加法 来计算

双向绑定

+=
*={{x1*y1}}
+={{x1+y1}}

双向绑定 使用插值表达式 获取结果 观察

x1:{{x1}} |||| y1:{{y1}} ||||| z1:{{z1}}
